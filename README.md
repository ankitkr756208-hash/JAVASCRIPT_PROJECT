# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe Game** built using **HTML, CSS, and JavaScript**.
This project demonstrates DOM manipulation, game logic implementation, and user interaction using pure JavaScript.

---

## 🚀 Live Demo

👉 https://tictactoeankit7562.netlify.app/

---

## 📌 Features

* ✅ Two Player Game (X vs O)
* ✅ Interactive UI
* ✅ Winner Detection Logic
* ✅ Draw Match Detection
* ✅ Restart Game Option
* ✅ Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
